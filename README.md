In this project, we obtained the monthly stock prices of two major Indian IT companies, INFOSYS and HCL Tech, over a duration of four years, starting from April 2021 to March 2025. We collected the stock prices for both the companies on the first trading day of each month during this period.

After collecting the data, we conducted a detailed time series analysis. We performed data visualization to find the underlying patterns and decomposed the series to identify various components such as trend, seasonal variations and random error components.

Subsequently, we tested both datasets for stationarity. Initially, both the datasets appeared to be non-stationary, which is a common scenario. To address this, we applied differencing techniques to transform the data into stationary form.

Following this, we used machine learning algorithms to build appropriate time series models for the datasets. Finally, we evaluated the performance of the models using various accuracy metrics to assess their effectiveness. We also validated the models and obtained fitted values. Using these models, we forecasted the stock prices for both companies for the next one-year time period.
